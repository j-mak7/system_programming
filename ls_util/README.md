# Утилита вывода содержимого каталогов

Этот проект является аналогом стандартной утилиты ls с поддержкой системных вызовов через C и ассемблера. Выбор осуществляется с помощью директив препроцессора на этапе сборки программы. Для утилиты было реализовано базовое поведение (без флагов) и поддержка флагов -a и -i.

## Возможности
- Вывод содержимого каталогов
- Поддержка стандартного ввода/вывода (stdin/stdout)
- Поддержка флагов -a и -i
- Два режима работы: C и ASM
## Команды сборки

Сборка с ASM-реализацией:

``` bash
make DFL=1
```

Сборка с C-реализацией:

``` bash
make
# или
make DFL=0
```

Очистка:

``` bash
make clean
```

## Использование

```
./ls [option]... [file]...
```
