# system_programming

### framebuffer_driver
Драйвер для работы с изображением (Framebuffer), обеспечивающий вывод изображения на экран.

### kernel
Лабораторная работа для взаимодействия с системной BIOS через интерфейс SMBIOS (System Management BIOS) на уровне ядра операционной системы.

### ls_util
Утилита для работы с данными на низком уровне, наподобие стандартной утилиты `ls`.

### uefi_bootloader
Лабораторная работа по созданию загрузчика для UEFI.
